# Playwright Automated Testing

Automate QA tests with Playwright

## File Structure

- `/tests` Contains all test cases to be run grouped by file like, `test-ga-page-props.js`
- `/tests-config` Contains specific configuration files for the tests.

## Installation and Setup

1. Install packages with `npm install`

## Running Tests

- Run tests in local env with `npm run test`
- A report is also output in the `tests-results` folder.
- Pass optional base URL: `BASE_URL='https://preprod.metro.co.uk/' npm run test`.
